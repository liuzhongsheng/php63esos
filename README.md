<h1>系统说明</h1>
<h3>目录结构</h3>
    |---App 应用目录
    |    |---Runtime 缓存目录
    |---Bigcow 框架目录
    |    |---Bigcow.php 框架入口文件
    |---index.php 项目入口文件

<h3>系统常量</h3>
    SYSTEM_PATH  系统目录常量
    APP_PATH     应用目录路径常量
    APP_DEBUG    调试模式默认false
    LIB_PATH     系统核心类库目录
    CORE_PATH    框架类库
    COMMON_PATH  公共模块目录
    VENDOR_PATH  第三方类库目录
    CONF_EXT     配置文件后缀
